This project is a single-page technical report webpage that presents an overview of global green energy trends.
It highlights renewable energy technologies, comparison data, and country-level renewable energy performance.
The webpage was built using basic HTML to demonstrate semantic structure and technical reporting in a web format.
It features the following:
1.Executive summary section
2.Green energy technology descriptions
3.Global energy comparison table
4.Four-country renewable energy comparison
5.Contact footer
